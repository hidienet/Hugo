[食道鉄道北海道] Day4：天金本店 - うにしゃぶ

今晚卜咗枱食癲
呢間天金本店已經有80年歷史㗎喇～

髙砂酒造 純米酒
帶著「今、選び抜かれたあなたへ」嘅旗號
無理由唔揀你嚟飲嘅…
味道超fresh又勁濃果香味，爽呀！

專登為咗呢兜鍋仔而訂枱（期待）

而呢兜仔就係望餐牌見到人気冧巴溫先嗌（期待too）

函館産松笠鯛刺身
反正今次旅程落唔到函館
就食返幾片嗰度嘅松笠鯛刺身
鯛魚就係鯛魚，香而不膩食幾多塊都唔厭

仲等緊仲等緊
睇睇佢啲火鍋料先…

有新鮮鯛魚、象拔蚌、帆立貝、海膽
就算而家即刻食都完全可以嘅類型

登登登凳：うにしゃぶ

特登嚟食嘅海膽しゃぶ啊！
拎海膽打邊爐喎，豪到爆嘅感覺
唔駛用好豪嘅價格就食到喇
仲要好新鮮好好食

濃味嘅海膽配濃郁嘅海膽湯底
簡直係人生嘅一大享受～

自分で造る卓上手造り豆腐
然後再試埋呀冧巴溫
上枱先煮成形嘅嫩豆腐

就咁加醬油就已經好好味

係啦，唔會唔試海膽豆腐呢個組合嘅

清晒～

郷土天ぷら
（ぼたんえび2尾・かに・サーモン・イカ・帆立・野菜三種）
鄉土嘢唔係食土㗎
有蝦有蟹有三文魚魷魚仲有帶子滿滿海洋風情
新鮮度爆錶，超～ 美～ 味～～～

あらごしみかん酒
即係好好飲嘅梅乃宿柚子酒嘅蜜柑版
非！常！好！飲！
忍唔住之後响BicCamera見到有蜜柑酒呢樣嘢托咗支返屋企

モッツァレラチーズのわさび醤油漬け
佐酒小食，wasabi醬油醃mozzarella
和式風味嘅醃芝士，正！

うなぎの骨（から揚け）
哎吔吔，其實呢度嘅鰻魚料理都好出色
不過實在太衝住個海膽鍋
太飽食唔到鰻魚肉就食送酒用嘅脆炸鰻魚骨
哇哇哇好大一兜食得好滿足呢～
