[神奈川衝撞旅] Day2：川崎大師平間寺

瞓夠起身都唔係好晏
應該只係因為去旅行好醒神同年紀冇關

搭個京急到川崎大師行個圈

一出站就會見到表参道牌坊

沿住啲造型街燈行過去就唔會蕩失路

川崎大師平間寺
而個寺响大治3年（1128年）創建
呢個大山門係入口
於昭和52年（1977年）11月開創850年記念落成
門左側有寺号碑（寺標）
係昭和59年（1984年）
弘法大師空海上人嘅1150年御忌時立

下面用嚟經過
上層係唔公開嘅經庫

大本堂

昭和39年（1964年）5月落成
祭祀本尊厄除弘法大師嘅

襯冇人走入嚟行吓（除鞋）

每朝都會响呢個堂進行祈願

堂内安奉咗稚児大師、救世観音像、不動明王、愛染明王、金剛界曼荼羅、胎蔵界曼荼羅

菊花紋章

中書院
前面舊石係茶筅塚

見到有洗淨嘅地方

因為前面係不動堂啊！
本身响明治23年（1890年）創建
於昭和39年（1964年）再建
裡面嘅本尊係不動明王

忠魂碑
明治42年（1909年）1月立碑

寺務所大本坊

走出西解脫門之後
又响至真門走返入寺園區

印度風寺廟藥師殿
昭和38年（1963年）創建

裡面安奉住弘法大師、不動明王、般若守護十六善神

依據生肖嚟整嘅「紀念品」

加埋條繩就搞掂

鐘楼堂
大晦日、6月10日の時記念日、8月6日広島原爆忌、8月9日長崎原爆忌同8月15日終戦の日都會敲鐘

全金身嘅釈尊像

穿過やすらぎ橋…

就到返中庭嘅中興塔（八角五重塔）
係昭和59年（1984年），弘法大師1150年御遠忌時落成
毎月第一個星期日同21日會開放內部供參拜

第五十五代横綱 北の湖敏満之像

遍路大師像
呢度嘅本尊弘法大師嘅石像

再一次經過西解脫門
今次就係真出走喇～
