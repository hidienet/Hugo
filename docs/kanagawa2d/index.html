[神奈川衝撞旅] Day2：キリンビール 横浜工場

正所謂「人生就是不斷的學習」…

キリンビール 横浜工場

咁唯有透過參觀工場嚟乖乖學習啦～

網上預約，再到場登記
然後就分發咗個翻譯機同Discovery Book

着住cup noodle去kirin
唔同性質唔算踩場啊！

襯離見学仲有啲時間
响等候區兜個round先

呢度有扭蛋

有得影自己樣印做啤酒盒錢罌紀念品

仲可以測試自己喜好而推介啤酒

夠鐘上堂喇！

先學啤酒三元素
有得睇吓聞吓玩上手（食埋都OK）

關於麦：梗係要同其他牌子比較吓

個工場好乾淨呢～

講解麥芽搾成麥汁嘅過程

呢間酒廠只會用第一搾嘅麥汁
咁同其他公司唔好嘥搾多次嘅有咩分別？

試飲咗就知只係第一搾會甜幾多倍囉
（個顏色都深好多啦）

釀好嘅啤酒就响呢個大輪裝起佢

一分鐘可以入到兩千罐㗎！

參觀無料之外仲有三杯飲（yeah～）
有三隻tap梗係會試晒佢！！！

第一杯係最平常見到嘅一番搾り生ビール
工場內飲，突然好飲咗 XD

配埋工場限定嘅亀田の柿の種 チーズ味
呢隻有kirin酒酵母嘅芝士外層同脆卜卜花生
好好食㗎！

第二杯一番搾り〈黒生〉
焙煎咗嘅穀皮而釀到黑黑的
呢隻苦味嘛… 唔啱我…

最後係呢到先飲到嘅一番搾りプレミアム
premium啤係用聚泡聚香氣嘅杯裝住
因為隻麥芽味… 好飲過前兩杯好多好多好多呢！

不過免費啤嘅代價係：20分鐘內要飲晒三杯！

雖然無料天堂唔駛入場費又有得飲

但唔代表唔會消費啊
（鬼叫啲「工場限定品」咁L吸引咩）

好耐冇試過happy learning呢～～～
