2018 年 1 月 16 日

[C'mon 関門！] Day2： 下関 → 門司

一早由小倉坐火車到下関
再坐巴士到唐戶市場

見到唐戶市場呢隻大大ふぐ…

就知道一定唔可以唔食佢
呢度除咗ふぐ仲有好多發出寶石光芒嘅魚生
從新鮮度到價格
係有種唔食就一定後悔嘅地步
this is round 1 only（因為另外仲有兩round ㅋㅋㅋ）

落雨 x 海邊 x 大風 x ふぐ
所以會飲ふぐひれ酒暖暖身
（酒裡面有燒過嘅河豚魚鰭）

响唐戶唔單止食超新鮮嘅海鮮
仲有一棟「幸福大魚缸」－ 海響館
… 一街之隔，啲魚魚有住截然不同嘅命運

不過又有想去吓唔同嘅館
咁就入手関門遊遊チケット
一句講晒夾埋就係抵睇嗰啲囉

樓下係睇企鵝

上層就睇関門海峽嘅魚
裡面嘅魚唔係俾人食係俾人睇嘅

弱者唔俾人食可能俾其他魚食啩…

ふぐ們，你哋想要自由定要命？
但係又好似唔到你哋揀呢…

（サントリー - デリカメゾン ＜フルーティな白＞）
出咗館有啲口渴
咁啱飲ひれ酒嗰陣有幾隻杯
可以開枝白酒分咗佢，呵呵呵～

飲完酒就上関門連絡船去門司
好短途又唔平，完全係遊客之選（反眼）

稀有嘅好處係可以响海上睇関門橋囉
（係天氣差啲浪大啲隻船拋啲之嘛）

到門司就去另一個館：九州鉄道記念館
幾啱我呢個偽火車控㗎！（心）

睇完車車之後（落雨冇得揸車車呀 TT）
就去食門司嘅焼きカレー @bear fruits

食完嘢跳去門司港地ビール工房
試吓當地工場啤酒！
and（包括但不限於）食嘢，任由幾粒鐘流走…

再搭返火車由「0哩」嘅門司港駅返小倉

當然，返去瞓之前都會兜去便利店買雪糕
今晚嘅佳麗係七仔crossover桔梗庵嘅桔梗信玄餅ワッフルコーン 
あああ… 好好味呀！

食飽飽又一日喇～
