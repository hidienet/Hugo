[Happy Days！鹿児島宮崎！] Day 2：霧島神宮（高千穗牧場牛奶 x 霧島綠茶軟雪糕、霧島茶）

去霧島唔係唔行吓霧島神宮呀嘛？

相傳創建於6世紀嘅霧島神社
之後霧島火山爆過幾次後
消失咗又重建幾次咁啦

唔講喇，慳返啖氣行樓梯上去啦

話咁快就到二の鳥居

吖～ 到咗展望所
天氣好好！連桜島 (中下偏左濛濛嗰座山)都睇到
（其實影呢張相係唔知嘅，只係人影我又襯吓熱鬧咋。）

客殿同御座所側跟有紅葉呢

然後就到三の鳥居

好喇好喇，終於到霧島神宮嘅本殿喇

未行到個本殿，就俾右手邊呢棵御神木吸引
呢棵杉有八百歲大㗎喇
宜家三十八米高
腰圍七十二米（好粗！）
好似話係南九州D杉嘅祖先

拜唔拜神都好
去要手水舍洗乾淨手清吓口

好多人祈福呢
希望大家都幸幸福福啦～

恋みくじ：要求戀祈放低二百就可以！

掛～～～

抬頭又望到紅葉

兜過隔離就係神楽殿
有法師幫人祈願呀接受結婚式嘅
收錢做嘢啦當然

華麗轉身向下走

到休憩所睇吓產物食・雪・糕
有幾特別？
原料係附近高千穗牧場嘅牛奶 x 霧島綠茶
根本就係特產中嘅特產，一次過試晒
連個雪糕筒都脆過人，好好味！

but哎呀哎呀要趕巴士
要跑跑跑跑跑落去
快速跑過呢條神橋

好高好大座嘅「 真・入口」大鳥居 

終於跑到落返嚟霧島市観光案内所
因為可以免費存放行李啊！
梗係拎返D嘢走先鬆得人啦

走之前㩒返杯自助免費霧島茶

好香好清帶微微甘味好好飲（多謝）
試過凍嘅茶（@雪糕）；再試呢杯熱嘅，好滿足～

時間good fit 到站等巴士

搭巴士走喇，byebye神宮（揮手）
