車厘茄啦蜜柑啦士多啤梨啦
全部都係鹿兒島產物

出發喇，瞓陣先 Zzz…

當然會有得落車啦，輕鬆到達熊本仲可以買埋當地特產哈哈

（呢幾日都唔係幾夠瞓…）
