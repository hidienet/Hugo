[錦秋京都高野山] Day1：KIX → 京都

凌晨機，清晨落機

著陸先買張KANSAI THRU PASS（3 Day Pass）
可以唔駛連續用行程可以輕鬆啲

関西空港 (鉄道) → 天下茶屋 → 淡路 → 桂 → …

到喇！嵐山 (阪急線)

天氣好好呢～

渡月橋

葛野大堰… 吸啖空氣都清新過人

ほくほく、お団子セット
放低行李就終於食早餐
@eX cafe イクスカフェ 京都嵐山本店
自己炭燒串団子，仲有杯京都唔可以冇嘅抹茶（呵呵呵）

食完就漫步人生路一陣

望吓人哋輕舟已過萬重山

行到過去就係時候食晏 @湯豆腐 嵯峨野
嗌支「招牌」清酒 嵯峨野，係伏見の地酒

湯豆腐定食
間舖得一個set，都只係得一個set

湯！豆！腐！
一個字：好食！

嵐電 嵐山駅

to 龍安寺駅
（好靚呀架電車）

文化遺產之一嘅石庭 @龍安寺

～小器～

裡面個殿

舎利殿 金閣 @金閣寺
太陽射正，金光閃閃呢！

不動堂

出咗金閣寺再上巴士
到世界遺產：元離宮二条城

国宝・二の丸御殿

行行吓都四點幾已經係黃昏樣

二條城前 → 太秦天神川 | 嵐電天神川 → 嵐山
…再去四条大宮

晚餐企响度食 @立ち呑み 庶民
