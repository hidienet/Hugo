[錦秋京都高野山] Day3：嵐山 → 高野山

清晨時分就起身出發…

六時未滿天都未光

飲返杯籮神熱辣辣咖啡

南海高野山ケーブル
嵐山(京福線) → 西院(京福線→阪急線) → 淡路 → 天下茶屋 → 橋本(和歌山県) → 極楽橋 →

高野山到着！

好好天 x 零星紅葉

成福院

brunch @御菓子司かさ國 （みろく石本舗かさ國） 
新鮮草餅超韌；甘酒饅頭更令人驚艷

食完又行吓，太舒服…

金剛峰寺
総坪数48,295坪，有排行

大玄関

鐘楼
万延元年（1860年）大火燒焦後元治元年（1864年）重建至今

蟠龍庭
2,340平方米嘅石亭，屬全日本最大嘅級別

入場包埋嘅麩焼き，坐低嘆吓

紅葉，遲啲仲會開到滿山紅

秋意濃濃

最紅嘅一棵 @不動堂

蓮池

高野山上嘅蓮燈

行夠，出去搵食先

精進料理 三鈷膳 @花菱
不枉排咗一粒鐘… 好好食！

食完都唔係去瞓晏覺呀

又向山進發

大門

赤松院
