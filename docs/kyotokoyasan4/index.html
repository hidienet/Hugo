[錦秋京都高野山] Day4：高野山 → KIX

呼吸住清新空氣…

…去迎接晨曦

1泊2食嘅寺廟食呢份簡便嘅早餐

慢慢嘆完就日出喇
秋天嘅景致真係百看不厭呢～

苅萱堂
食飽check out完擺低行李就散吓步
呢個堂裡面啲木雕好靚呀呀呀～

散策

女人道

向下望

响山到上上落落

遠眺連海呀四国都望得到（真係德島 XD）

樹～

女人堂
以前女人只可以去到呢度
唔俾再上山㗎喇

就係呢度，女人止步。

而家嘅年代當然可以繼續行

行落山

纜車（心）

行落到南海電鉄極楽橋駅

極楽橋
早登極樂～（大誤！）

行夠就搭返纜車上山先（夠晒hea）

精進料理 lunch @中央食堂 さんぼう

食完再兜兜吓見到越嚟越多人…

有隊女choir响度喎

僧侶們浩浩蕩蕩進場

一路行一路飛啲嘢出嚟

書籤咁嘅樣，聽講係寓意錢嘅嘢！！！

原來係咁啱撞到嘅「錦秋大伽藍お練り法会」

着晒正裝好華麗啊

同偶像團體出巡一樣
好多人追住影（包括我）

收隊

都執到唔少錢㗎！（發達仔）

食埋杯柿就走喇～

落返山，去大阪先去Spa World浸嗰浴
再匯合朋友居酒屋一吓
然後坐夜晚機返香港lu…

～完～
