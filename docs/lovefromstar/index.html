活了那麼多年累積了不少財富與地塊（土豪無誤），愛收藏古董（其實都是自己跟朋友送的的東西）、有天文望遠鏡看星空（因為就算超好的視力也看不到家鄉惟有這樣）

這個「長得帥的笨蛋」（by劉在石@running man），笑容可愛加說話時會嘟嘟嘴只能以超萌來形容，愛演哭戲也是個愛哭寶寶（請參考綜藝節目taxi）
