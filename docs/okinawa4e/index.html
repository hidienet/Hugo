[琉球戀吃活樂遊] Day4：海援隊沖縄

有甚麼比琉球食king Eilly的200円一杯オリオン生サザンスター更吸引？

就是一杯オリオン生サザンスター只是100円
加上全店食物半額祭（半價！）

好！吃得狼死，START！！！

島らっきょ塩漬け
沖縄種植得蕎頭，以鹽淺醃
有點鹹，不過生蕎頭蠻辣
怕辣的我不能直接吃，只能拿來配酒才能吃

熊本馬刺
即馬肉刺身，馬是跑跑動物
所以沒有太多脂肪，肉質紮實肉味香
比牛肉好吃很多很多！

グルクン唐揚げ
沖縄獨有的魚，上次在花笠食堂吃煮的
這次終於試到炸的，香脆帶鹹的炸魚
裡面的肉質鮮甜，骨頭也是脆脆的

ハツ
喝啤酒來串串燒也不錯
雞心又香又軟又爽又彈牙

砂肝ゆず胡椒炒め
砂肝不是肝，是腎
柚子胡椒小辣而清爽
腎很爽脆，咬落爆汁，好吃！

ロ～ングバケットピザ
75cm有多長？看那隻手再對比對比
pizza底不是一般麵團
是baguette，烤起來超～ 鬆～ 脆～
雖然上面是最普通的番茄芝士煙肉等
但最簡單的就是最好的（baguette位加了很多分！）

干しイカ炙り
又好玩又新鮮美味的乾魷魚！
自己以炭烤～

無論在那個地方，烤魷魚都會很香
但吃起來就不是那裡都好吃
（尤其那些賣韓國魷魚乾的攤）
不過這個魷魚乾非常濃味
而且又帶點鹹，佐酒一流！（加上好玩！）

ゴーヤーサラダ
吃沙律，炸苦瓜片薄薄脆脆的
菜很新鮮很甜，沙律汁酸甜很開胃
（吃了那麼多還要開胃？）

やわらかてびち
豬手炆得又腍又滑
黏黏的好像一直在幫我補充骨膠原那樣

パリパリチーズスティック
炸芝士春卷，外香脆內流心
繼續送酒之用！

あつあげ豆腐
吃下感覺像臭豆腐的外脆內軟
不過不是臭的，因為是沒有發酵的硬豆腐
很喜歡這道豆腐！

イカポッポ焼き
再來一條燒魷魚，一點都不韌
新鮮嫩滑，醬汁又不會很濃

吃吃喝喝的四個小時
平均一個人也不用2300円
呵呵呵呵呵呵呵呵呵呵呵呵呵呵～
