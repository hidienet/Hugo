[臺中怒食遊] Day6：鹿港天后宮前的大蚵特蚵蚵蚵蚵

超「蠔」的晚餐～ （不要拿錯辣椒醬當啤酒來喝喔） 先來一份蚵仔煎 台式的蚵仔煎還是比潮式的腍身點 但蚵仔超鮮這點足以禰補一切 蚵仔粥也是is a must 潮州粥的粥底 有蚵粥就不選蚵仔湯 蛤仔湯很清甜的也不錯 怕不夠飽肚就要了個炒米粉 當然是有蚵在中的 也有吃炒菜，飲食要均衡 據說很有特色的蚵嗲 其實是一大舊炸物裡面有韭菜蔥粒蚵仔這樣 不是不好吃只是不對我口味而已 另，也有點了炒蛤蝲 但對於喜歡豉椒炒蜆的我 就覺得這裡的味道太淡 （連我這個淡味人都認為淡呢） 這碟川燙醋蚵 我一粒接一粒吃吃吃入口 喔～ 停不了，好愛這個（心）

[臺中怒食遊] Day6：鹿港牛舌餅
