[保加利亞醉遊記] 遊覽篇：Tsarevets

Veliko Turnovo作為保加利亞第二帝國的首都（1185-1396） 當然不能少了城堡這樣的東西 從民宿的露台都可以看得到 （照片右邊中間一高一矮的建築物） 看似好遠對吧！ 但我們是步行過去的喔（走走走） 鏡頭一轉，才能拍到護城牆 本身圍住Veliko Turnovo的Yàntra River 自然成為護城河 差不多到入口 只能走這條路要入城 哈哈哈～ 保加利亞獅子又來了 再說一遍：不論出入，也只有這一條路 舉頭已看到較近的一高一矮 被一堆城牆包圍住 向後拍，看到之前經過的教堂同歷史館 （所以已經喝了瓶「歷史雞精」） 也能看到住的地方（照片中間山上位置） 有素人的比例，看到護城牆有多高嗎？ 真正的戰車 害我一直認為這種戰車是中國的
