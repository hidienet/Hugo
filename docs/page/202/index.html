[seoul time] Day6：豐川鰻魚 풍천장어

豐川是位於韓國全羅北道南原市的河川 也是個以鰻魚聞名的地方 （因為那裡是河水同海水的交匯處） 這家叫鰻魚店也是以豐川풍천的名稱命名 廢話少說，先吃！ 烤河鰻 以糖、醬油、蒜、生薑等調味 在板上面燒，樣子似日本烤鰻般 但食法就是韓式地包菜同吃 鰻魚入口軟滑即化 醬油味道偏甜而鰻魚不算很腥 除了配薑包菜之外就好像驚喜不大 不過這個送酒一流的 脆脆炸饅魚骨小吃就不錯～ 辣湯？ 原來是拿了海鰻去做湯 （yeah，完全韓式做法） 辣湯底裡面還有海鮮（蟹、蜆乜乜乜） 不算太辣的湯卻是超鮮甜 海鰻肉非常彈牙又香 本身的味道不會被辣味蓋過 證明那是極之新鮮的貨色 好好吃呢～ -———————————————- 延伸閱讀：seoul time（7D7N）
