[保加利亞醉遊記] Day13：Varna to Sofia

要從Varna搭火車回Sofia了 這裡的火車站 紅 x 白的舊建築好漂亮 （下面會告訴你甚麼叫做不漂亮） Varna是沿海城市 掛畫所畫的就是以前很多船那時吧 今晚搭乘的火車 這跟之前Sofia去Plovdiv坐的一樣 不過今晚是過夜車 有床的車卡是漂亮一點的 （同一條車，只是車卡不同） 一卡裡面有三張床 樣子也算不賴 （這是普通等級的，頭等的會再好一點） 好好梳洗，睡醒清晨就到Sofia 告訴你甚麼叫做不漂亮的火車站 就是在Sofia這裡 又是大大舊毫無美感的共產式建築 裡面還要是修葺中 有8-9成的光管都是壞掉 （但wifi是正常沒有缺少的） 上次從Plovdiv回來是夜晚 整個地方又殘又舊又暗又臭

[保加利亞醉遊記] Day13：Бистро Цветното
