[seoul time] Day6：韓牛坊 한우방

又是睡到要吃brunch的時間
去烤頭高・級・韓・牛吧！

要 吃就要吃最好的
全部都是1等A+的高級韓牛外
不選餐，直接單點餐牌最貴那些
土豪叫法卻是貴精不貴多
左邊是鲜排骨 생갈비
右上是横膈膜肉 안창살
右下是牛里脊 생등심

別以為擺好的烤盤就是用gas爐
爐火是以江原道的橡樹木炭做底
木炭的清香就會滲進肉中使其更美味
當然吃韓燒少不了配大量的菜
加上小碟小碟的kimchi配菜
還有泡生洋蔥的醬油
同最能發揮韓牛最極致美味的麻油鹽
當然是有專人燒，這次是店長呢

先燒牛里脊 생등심

反了反了

再剪開就OK
脂肪不多肉質夠韌又有嚼勁
卻不會有渣的超好吃

然後是鲜排骨 생갈비
脂肪同肉分佈平均
肉味香醇又帶有油花
甚麼調味都不需要燒好就吃很美味

稱為最好的肉 － 横膈膜肉 안창살
（擱在一旁的是排骨舊骨）
無脂肪的筋位肉韌度一流
外面烤香裏面juicy，簡直是極品～！

吃完是飽而不是肥漏漏的，真好（超讚）
