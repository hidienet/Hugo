[食の四国ツアー] Day2：丸亀城

食完烏冬就行去現存12天守中
最細天守 x 最高石垣嘅丸亀城

响亀山公園裡面
抬頭望已經係好高好高嘅城牆
同好細好細粒嘅天守

於慶長2年（1597年）建成
係県指定文化財亦係国指定史跡

大手一の門
莊嚴嘅櫓門
主要部分都係以櫸木造

大手二の門
同大手一の門一齊起嘅
屬於高麗門形式嘅門
呢道門同時睇到上面丸亀城個樣

扇の勾配
keep得好好嘅石垣
啲曲線批得好靚

上到一半已經睇到周圍嘅景

月見櫓の跡望出去嘅景係一群連住嘅山

行出去睇吓先

雖然天色唔係好清
都輕輕睇到瀨戶內海同讃岐広島本島牛島

就上到頂喇

啊～見到天守喇

本丸

丸亀城 天守閣
小小一座約15米高
而座山本身都標高約66米
周圍嚟講當然高，個景可以一覽無遺

唔覺得入去睇景同出面有咩大分別
又對裡面嘅建築冇好大嘅興趣
就冇走入去行嘅衝動，兜多陣就落返去

日晷，陰陰天唔係睇得好清楚

二の丸嘅二の丸井戸
二の丸已經冇咗，得返個井
然而佢係日本最深嘅井戶
約水深30米以上

落返地面

かぶと岩
京極氏時代嘅火山岩

玄関先御門
响另一邊門走
係舊藩主居館嘅玄関

過埋護城河

個城仔，真係得細細粒（哈）

出返嚟通町
架巴士都係細細粒，好得意～
