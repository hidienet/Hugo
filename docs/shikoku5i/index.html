[食の四国ツアー] Day5：大観覧車くるりん

JR：宇和島 → 松山（再行去大手町）
伊予鉄道：大手町 → 松山市

再上埋呢個大観覧車くるりん嘅摩天輪
全部都係JR pass包晒
本身對摩天輪冇乜迷戀嘅感覺
咁既然免費又有時間剩
搭吓百貨公司頂樓嘅摩天輪睇吓個松山景都唔錯吖

個摩天輪座落於高島屋9階

地下連摩天輪，離地85米（最高點）

可眺望聽朝去嘅松山城

個天台有坊っちゃん列車（少爺列車）個樣，哈哈

雖然厚雲咗啲

但唔影響個景觀啦

轉一個圈大概15分鐘
睇完又唔想行百貨公司
咁就唔再留戀呢棟大廈喇…
