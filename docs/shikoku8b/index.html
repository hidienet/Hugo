[食の四国ツアー] Day8：高知の日曜市（土佐の街路市）

早早起行就係為咗要快人一步嚟日曜市

（街頭）先快快行一次先

遠睇已經被個極火紅吸引
以為係啲唔知咩花
行埋就見到原來係原棵辣椒

紅橙黃色鮮艷搶眼嘅生果同番茄們

圓咕碌一個個波咁嘅土佐文旦同柑

一條路千三米多啲
係全日本最長大嘅露天市集
呢個只係開星期日嘅市集
由元禄3年（1690年）至今已經有三百年歷史
而家有四五百個攤檔
有嘢買又有嘢食好豐富㗎

蕃薯等等根菜薯類係好食（仲好抵）
不過太重喇搬唔到返去

乾物香料茶業一袋袋

新鮮閃令令魚皮顏色嘅魚乾們
冇咩腥味真係好想買好多好多返屋企

健康成長嘅菜肥肥大大舊
仲要明明種响田度但擺出嚟個個都好乾淨咁
可惜太大太重又冇理由托佢哋返去嘅

唔揀新鮮菜都可以揀醃菜
然而本身唔太鍾意食鹹菜啦

紫蘇梅就好OK，大愛大愛（心）

哎吔吔，啲手造糕點好吸引咁喎…

舊醬油味付鰹魚好正就梗㗎喇
個鰹魚胃造嘅酒盗更更更吸引呢！

又平又靚仲大大袋
體力有限冇可能拎走啦

買唔到又冇得即食嘅海鮮們（淚）

出產鰹魚嘅地方
梗係唔少得鰹魚們同其製品

高知特產帽子麵包ぼうしパン

行到街尾附近就係non-food嘅地帶

～花～

（街尾）
夠早先可以唔駛迫人下行完一round

開始行去睇吓有咩入手喇

快步行咗一轉
大概就知咩嘢大概咩價位
同埋咩檔位啲嘢啱買

晏啲啲(9am)就已經咁水洩不通㗎喇
即係都係時候走喇
（已經兜完3 round，成條街行咗6次唔駛再磨啦）
