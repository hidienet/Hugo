[菠蘿 ‧ 酥time] 李製餅家 鳳梨酥

大房是李製餅家、二房的李鵠餅店、三房的李儀餅店 其實李鵠才是李源祥在1882年清光緒時期創立 當然，沒有多少有名的家族是不分家的 古老古老的包裝～ 長方扁身的鳳梨酥 外皮香味濃，咬落酥脆 皮層較薄所以一點都不覺得油膩 冬瓜蓉做的鳳梨餡酸甜適中 微漿的內餡跟薄酥的外皮 呀～ 一次食三四個都不覺得膩 雖然我很愛較新的土鳳梨酥 但這份古早味也令我欲罷不能呢～ （* 個人誠意推薦！）

[菠蘿 ‧ 酥time] 李儀餅店 鳳梨酥

在九份等巴士時殺了的一盒（笑） 這個包裝有夠老土呢～ 同包裝盒子一樣 鳳梨酥都是四四方方起角的 金黃色的酥皮面層焦得很平均 但酥皮較厚實，原有的奶酥味也太淡 吃起來感覺有點喉嚨乾乾 內餡比較漿，直接開封冷吃的話 還會一舊舊糾結在一起 如果叮一下微熱會好一點 不過就會是糖漿溶了的又漿又黏 這時候的酥皮更散更碎 總之就是順得哥情失嫂意（悶吶）

[菠蘿 ‧ 酥time] 港記酥皇店 鳳梨酥

因為之前有個舊同事一嚷著要買這個 所以好吧我也試一下（不蝕底） 娘爆的包裝（！） 叫「港記」因為始創人是香港人 但也是做傳統recipe的鳳梨酥 加上據說是全手工製造的 聽起來好像不錯啦～ 不太厚身的金黃色外皮一咬就化開 大方地塞了滿滿的鳳梨餡 這是彰化的鳳梨耶 所以還是帶出微微酸味跟纖維 其實覺得沒有那個舊同事說的那麼驚豔 但也算是不錯吃的選擇而已～
