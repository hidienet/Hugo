[昆明飽hihi] Day2：蔼若春

在雲南吃汽鍋雞是常識好吧！ 結果當然是煲湯油到爆啦 XD 汽鍋雞是以這個中間有條喉的建水土陶汽鍋 以雞及其他材料，不加水蒸兩三個小時的 所以那層是全！雞！油！ 不過隔開那層油之後 就會先聞到超香的濃厚雞湯味（十盒史雲生的味道） 在黃金色的湯頭裡面品嘗到純粹鮮甜的雞湯 雞的骨肉分離（！）而且雞肉嫩嫩滑滑的 到雲南不喝這湯絕對走寶就是！ 焖饭 好大一鍋，很香也很好吃 不過真的很油膩（飽肚飽肚） 接下來，猜一下以下的東西是什麼 白色一盆，點的「醬」是桂花糖 再一個角度 白白的，晶瑩剔透 吃下是爽脆，沒什麼味道所以加糖就會甜 它，不是淮山 它，不是玉竹
