[ドキドキ鳥取] Day1：米子（米子鬼太郎空港、米子駅、大山黒牛処 強小亭）

臨出發前一星期先話廚叉可能釘牌喎… -_-

米子鬼太郎空港到着，呵呵

嗨（揮手揮手）

印仔time

巴士係2-3倍JR嘅價格…

所以就慢慢拖行李去搭JR
（米子空港駅／「べとべとさん」駅 → 米子駅／「ねずみ男」駅）

哈哈哈！一上已經係鬼太郎列車
抬頭望天花都係鬼太郎同目玉親父

啲凳都係鬼太郎同爸爸

落車

繼續影影影

呢程係兩卡車，另一卡係目玉おやじ列車

米子駅／「ねずみ男」駅

個月台都有好多角色啊啊啊

反正唔趕時間，慢慢影

境線全部都有鬼太郎角色名做站名㗎

先告別呢個0號月台，去酒店check in

然後就行去搵食

話說呢條係午後六時嘅商店街
荒蕪到一個點…

終於嚟到大山黒牛処 強小亭

食大山牛梗係配大山啤

肚餓就唔諗太多，揀set啦

燒嘅shabu牛

大山牛… 無咩㗎，拎過日本第一黑毛和牛獎之嘛…
燒完之後油花咇出香而不漏咁

燒suki牛，將すき焼き燒嚟食

燒呀燒，再加咗個內臟盤

本日の希少部位四種（邊打字邊流口水）

好在燒牛嘅時間唔漫長

個set仲有牛肉炒飯同超讚嘅牛骨湯（&甜品）

好飽呀黐線！！！

飽還飽，都係想食雪糕冰品
有Glicoアイスの実 いちご，仲係mini sele迷你版
好似飲緊新鮮沙冰咁清新

既然食咗包半份量嘅即係可以再擺嘢落肚
明治 エッセル スーパーカップ レアチーズケーキ
lemon cheesecake個檸檬味同cream cheese味都幾出
再加埋有粒粒餅碎係有啲似放咗入冰格嘅cheesecake喎～

好！食飽瞓 Zzz…
