[ドキドキ鳥取] Day3.2：由良 → 倉吉 → 三朝温泉（ラーメン幸雅、三朝温泉後楽）

離開由良，我坐上向鳥取方向嘅區間車。

等咗粒幾鐘終於見到你（好感動）

好大碌彩虹

（由良駅 → 倉吉駅）
係ひなビタ♪ 啊～

お倉 + お吉 = 倉吉

ラーメン幸雅
一出站一直行行行六分鐘就到
就算係拖住行李都唔辛苦

加上食到呢碗牛骨ラーメン
真係點都值得

食完見到啲雨大粒咗
咁唔去遊覽倉吉住直接搭巴士去三朝温泉

三朝温泉後楽
一落巴士過馬路就已經係門口
會唔會太太太方便呀？

勁vintage嘅門同個匙牌

好大嘅空間

welcome餅係元祖因幡の白うさぎ
好好食，一定要買返去再食過

既～ 然～ 比預定時間早咗嚟度
就落去浸次溫泉先啦

浸完梗係要食晚飯啦
塊鳥取黒毛和牛好大塊啊
個餐除咗呢堆，仲有天麩羅茶碗蒸碗物
釜飯係蟹同山野菜
當然唔少得生果同甜品（飽）

三朝正宗：嚟三朝係要試吓地酒嘅 

食到好飽唔浸得溫泉住
落街周圍兜吓先

三朝橋樣水渠蓋

古舊嘅菓子店てりふり屋

鳥取紙燈籠

泉娯楽場
超！懷！舊！嘅機舖

三朝温泉嘅鼎鼎大名河原風呂
係露天裸湯，然而真係有人浸緊（影完先知罪過）
因為冇諗到啲人會天寒地凍落住雨都要擔住遮都要浸㗎嘛…

過咗橋對面，呢邊先有超市仔嘛

行完街返到去，就見到床已經鋪好

大山おいしい牛乳モナカ

外形可愛，牛奶味濃郁好好食～

食完唞吓梗係再去浸過啦！
今晚一定好好瞓 Zzz…
