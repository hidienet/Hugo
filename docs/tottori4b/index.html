[ドキドキ鳥取] Day4.2：倉吉 → 鳥取（鳥取二十世紀梨記念館・なしっこ館、味暦あんべ、炉端かば 鳥取駅前店）

繼續倉吉行…

鳥取二十世紀梨記念館・なしっこ館
由倉吉白壁土藏群行12分鐘就到

又利用まんが王国とっとり満喫周遊パス換入場券
再計C9數：料金300円；用JR West pass半價 = 150円；鳥取藩乗放題手形bus pass無得減
慳返幾百咁啦都～

睇吓日本以至世界國地唔同梨嘅品種

梨と生きる「二十世紀梨」ものがたり劇場
呢個劇場好好睇，啲人仲會郁㗎勁詭異（！）

二十世紀梨の巨木
兩層高，直徑20米地底根部都有5米
差唔多成百歲㗎喇佢

其實個館係啱小朋友玩嘅哈哈

以前木箱啲標籤

而最最最吸引嘅當然係試食梨

切好雪凍慢慢試

未諗到有咩好買…

搭返巴士返JR倉吉駅

返倉吉観光物産館 長生堂拎行李

搭特急 京都
但係個pass只包到去下一個站

咁其實我又只係去下一個站啦（倉吉 → 鳥取）

啲座位勁舒服

開始日落

20分鐘就到鳥取駅

去酒店check in擺低行李先

落街食晚飯

好彩晌開門前10分鐘到門前排
只係排兩三個位
再過幾分鐘條龍已經好長lu…

味暦あんべ
終於開店 & 輪到

自家製 果実酒 レモン
有唔甜同正常甜，兩樣都試吓

親がに丼
蟹！蟹！！蟹！！！蟹！！！！蟹！！！！！
chisin㗎～～～～～～～～～～（尖叫）

就算配飯嘅がに汁都絕不欺場
滿滿蟹籽嘅半身蟹，仲可以任添飲啊！

食飽飽就去行吓街
行幾個字荒蕪嘅路如果唔係靠Google神
真係無可能去得到呢

哈哈哈係冇藥品連眼藥水都冇得賣嘅Don Q
OK嘅有護膚品吖都…

再行多荒蕪路段嘅廿分鐘去依昂
有藥品喇終於，不過都只係得少少無咩行啦

話晒响Don Q都買咗一堆嘢
由依昂行返酒店10分鐘咁放低嘢先啦

落街食宵夜 @炉端かば 鳥取駅前店

山陰八爪魚呀菇呀當然唔少得

仲有都係山陰山芋白蘿蔔
嗱炸河豚，河豚喎我係冇免疫力一定要點嘅

呢個燒雞皮正呀！
唔會燒到超乾身但係又唔會唔留脆脆位

食完嘢，唞吓就落樓浸酒店個溫泉
雖然唔係天然而係人工溫泉吖
有得浸就已經好好啦～

飽得滯+今日有食過
今晚唔食雪糕喇
