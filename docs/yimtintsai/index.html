鹽田繪 - 樟樹 Camphor Tree
鹽田仔係客家 x 天主教嘅村落
係香港絕少見到「冇天后廟得教堂」嘅島嶼
而天主教其中一樣特色就係彩繪玻璃窗
呢度都有，仲係客家風情話嗰種

鹽田，晏啲先落嚟睇你

食柴火土窯雞嘅餐廳冇開呀
得返貓貓响枱面瞓覺

荒廢加上山竹嘅蹂躪
建築物都呈現住頹廢嘅感覺

成個島得村上唯二居民間屋有人氣㗎咋

澄波學校
建於1846年，辦學至1997年
（有貼紙就可以直接入去參觀）

其中一個校舍改建為鹽田梓村文物陳列室

展品分為農業、工業、日常生活同課室四個部分

話新唔新話舊又講唔上好舊咁啦

鹽田梓聖若瑟小堂 | St. Joseph’s Chapel
响澄波學校隔離，係鹽田梓嘅地標

由奧地利聖言會會士聖福若瑟 (St. Josef Freinademetz)嚟傳教嗰陣興建
1840年祝聖，再响1948年、1962年同2004年進行過修復工程

响2005/6年獲聯合國教科文組織（亞太地區）文化遺產保護獎
而古物古蹟辦事處都將佢評為三級歷史建築

天主教堂一定有嘅鐘
不過眼看手勿動唔好手多多去敲響佢喎！

玫瑰窗

百年老樟樹
响客家人嘅習俗中，如果生咗個女
阿爸阿媽就會種棵樟樹
等佢大個咗就劈樹整個樟木櫃嚟做嫁妝
呢棵樟樹就係嗰陣種落嘅
冇劈落嚟唔知係咪條女嫁唔出啦

又話說當年日本侵華嗰陣
啲日話軍要斬咗棵樹，咁當時嘅村長為咗保護棵樟樹
就同日軍講要斬咗棵樟樹就斬咗佢先啦！
結果，棵樹到依家仲响度而村長嘛…（唔知喎）
